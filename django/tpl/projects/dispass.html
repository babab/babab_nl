DisPass Software project

Python   Security   Command line   GUI  

A passphrase generator for GNU/Linux, *BSD, MacOS X and Windows. Generate unique passphrases formed from a master password and a label, helping you get rid of the bad habit of using a single password for multiple websites. When using a different passphrase for every website, the chance of abuse of your password on other sites (when a website leaks it) is eliminated.

Dispass is a console application, but also has a simple graphical interface. It is released under an open source BSD-like license known as the ISC license
